# Description

Containing examples for the [Confluence REST API](https://developer.atlassian.com/server/confluence/confluence-rest-api-examples/) using shell scripts.

# Requirements

- Linux / Unix Shell
  or
- Windows Shell
- Cygwin + curl
    - Intune-Client or
    - https://curl.se/windows/
